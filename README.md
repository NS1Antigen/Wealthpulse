# WealthPulse PWA

Private local-only asset tracker.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Recommended: Vercel or Netlify.

## iPhone PWA install

Open the deployed link in Safari → Share → Add to Home Screen.

## Important

Data is stored in browser localStorage. Export JSON backup before changing phone or clearing browser data.
